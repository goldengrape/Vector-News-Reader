# Vector | 高精度ニュースレーダー

[English](./README.md) | [简体中文](./README_zh.md) | [한국어](./README_ko.md)

**Vector** は、AI駆動型のインテリジェントなテックニュースアグリゲーターです。あなたの認知的嗜好を学習し、ノイズを除去して、シグナル強度の高い情報のみをお届けします。

エンゲージメント（クリック数）を最適化する従来のRSSリーダーやアルゴリズムフィードとは異なり、Vectorは、あなたとAIが共同で作成する透明性の高い「自然言語フィルター」に基づいて、**情報の密度と関連性**を最適化します。

## 🚀 主な機能

*   **グローバルテック監視**: トップクラスのテックメディア（TechCrunch、Wired、Nature、IEEE Spectrumなど）からニュースを集約します。
*   **多言語サポート**: タイトルと要約を自動的に好みの言語（**日本語、英語、中国語、韓国語**）に翻訳し、高速なスキャンを可能にします。
*   **プリファレンスラボ (調整)**:
    *   生のニュースフィードに対して「興味あり/なし」を投票します。
    *   あなたの情報代謝メカニズムを分析した詳細な**心理的ペルソナ**を生成します。
    *   あなたにとって何が「シグナル」で何が「ノイズ」かを明確に定義する**自然言語フィルター (システムプロンプト)** を構築します。
*   **スマートリーダー**: カスタムフィルターを受信ニュースに適用し、AIが特定の記事を選んだ*理由*を説明します。
*   **プライバシーファースト**: すべての設定とAPIキーはブラウザの **LocalStorage** に保存されます。外部データベースは使用しません。
*   **BYOK (キー持ち込み)**: 独自の Google Gemini API キーをサポートし、レート制限を回避します。

## 🛠 技術スタック

*   **フロントエンド**: React 19, TypeScript, Tailwind CSS.
*   **AI エンジン**: Google Gemini API (`@google/genai` SDK).
    *   モデル: Gemini 3.0 Flash (デフォルト), Gemini 3.0 Pro (オプション).
*   **データソース**: CORSプロキシ経由のリアルタイムRSSフィード。

## 📦 インストールとセットアップ

1.  **リポジトリのクローン**:
    ```bash
    git clone https://github.com/your-username/vector-news-radar.git
    cd vector-news-radar
    ```

2.  **依存関係のインストール**:
    ```bash
    npm install
    ```

3.  **開発サーバーの起動**:
    ```bash
    npm start
    ```

## ⚙️ 設定

安定した利用（特に大量のフィルタリングを行う場合）のために：

1.  トップナビゲーションバーの **設定 (歯車アイコン)** をクリックします。
2.  あなたの **Google Gemini API Key** を入力します。[Google AI Studio](https://aistudio.google.com/app/apikey) で取得できます。
3.  **出力言語** を選択します（日本語を選択可能）。
4.  推奨モデルを選択します（速度とコストのバランスが良い Gemini 3.0 Flash を推奨）。

## 📖 使用方法

1.  **ラボモード (Lab Mode)**:
    *   生のニュースフィード（自動翻訳済み）を読みます。
    *   少なくとも10個の記事を「興味あり」または「興味なし」としてマークします。
    *   **「フィルターを生成 (Generate Filter)」** をクリックします。
    *   生成されたペルソナとフィルターロジックを確認し、**「有効にして読む (Enable & Read)」** をクリックします。
2.  **リーダーモード (Reader Mode)**:
    *   Vectorは、生成されたロジックを使用して、新しいニュースバッチを自動的にフィルタリングします。
    *   「一致理由」タグにカーソルを合わせると、AIがその記事を選んだ理由が表示されます。
3.  **微調整**:
    *   AIが見落としをしている場合は、リーダービューでシステムプロンプトを手動で編集できます。
    *   興味が変わった場合は、ラボに戻って最初から再トレーニングできます。
