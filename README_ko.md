# Vector | 정밀 테크 뉴스 레이더

[English](./README.md) | [简体中文](./README_zh.md) | [日本語](./README_ja.md)

**Vector**는 AI 기반의 지능형 테크 뉴스 애그리게이터입니다. 사용자의 인지 선호도를 학습하여 소음을 걸러내고, 신호 강도가 높은 정보만을 전달합니다.

클릭률(Engagement)을 최적화하는 기존의 RSS 리더나 알고리즘 피드와 달리, Vector는 사용자와 AI가 함께 만드는 투명한 "자연어 필터"를 기반으로 **정보의 밀도와 관련성**을 최적화합니다.

## 🚀 핵심 기능

*   **글로벌 테크 모니터링**: 최상위 테크 미디어(TechCrunch, Wired, Nature, IEEE Spectrum 등)의 뉴스를 실시간으로 집계합니다.
*   **다국어 지원**: 제목과 요약을 선호하는 언어(**한국어, 영어, 중국어, 일본어**)로 자동 번역하여 빠르게 스캔할 수 있습니다.
*   **선호도 연구소 (보정)**:
    *   원시 뉴스 피드에 대해 "좋아요/싫어요"를 투표합니다.
    *   사용자의 정보 소비 습관을 분석한 깊이 있는 **심리 페르소나**를 생성합니다.
    *   무엇이 "신호"이고 무엇이 "소음"인지 명확하게 정의하는 **자연어 필터(시스템 프롬프트)**를 구축합니다.
*   **스마트 리더**: 생성된 필터를 수신되는 뉴스에 적용하고, AI가 특정 기사를 선택한 *이유*를 설명합니다.
*   **프라이버시 우선**: 모든 설정과 API 키는 브라우저의 **LocalStorage**에 저장됩니다. 외부 데이터베이스를 사용하지 않습니다.
*   **BYOK (키 직접 사용)**: 사용자의 Google Gemini API 키를 지원하여 속도 제한을 피할 수 있습니다.

## 🛠 기술 스택

*   **프론트엔드**: React 19, TypeScript, Tailwind CSS.
*   **AI 엔진**: Google Gemini API (`@google/genai` SDK).
    *   모델: Gemini 3.0 Flash (기본), Gemini 3.0 Pro (옵션).
*   **데이터 소스**: CORS 프록시를 통한 실시간 RSS 피드.

## 📦 설치 및 설정

1.  **저장소 복제**:
    ```bash
    git clone https://github.com/your-username/vector-news-radar.git
    cd vector-news-radar
    ```

2.  **의존성 설치**:
    ```bash
    npm install
    ```

3.  **개발 서버 시작**:
    ```bash
    npm start
    ```

## ⚙️ 설정

안정적인 사용(특히 대량의 필터링 시)을 위해:

1.  상단 내비게이션 바의 **설정(기어 아이콘)**을 클릭합니다.
2.  **Google Gemini API Key**를 입력합니다. [Google AI Studio](https://aistudio.google.com/app/apikey)에서 무료로 발급받을 수 있습니다.
3.  **출력 언어**를 선택합니다 (한국어 선택 가능).
4.  선호하는 모델을 선택합니다 (속도와 비용의 균형을 위해 Gemini 3.0 Flash 권장).

## 📖 사용 방법

1.  **연구소 모드 (Lab Mode)**:
    *   (자동 번역된) 원시 뉴스 피드를 읽습니다.
    *   최소 10개의 뉴스에 대해 "관심 있음" 또는 "관심 없음"을 표시합니다.
    *   **"필터 생성 (Generate Filter)"**을 클릭합니다.
    *   생성된 페르소나와 필터 로직을 검토하고 **"활성화 및 읽기 (Enable & Read)"**를 클릭합니다.
2.  **리더 모드 (Reader Mode)**:
    *   Vector는 이제 생성된 로직을 사용하여 새로운 뉴스 배치를 자동으로 필터링합니다.
    *   "일치 사유" 태그에 마우스를 올리면 AI가 해당 기사를 선택한 이유를 볼 수 있습니다.
3.  **미세 조정**:
    *   AI가 놓친 부분이 있다면 리더 뷰에서 시스템 프롬프트를 수동으로 편집할 수 있습니다.
    *   관심사가 바뀌면 언제든지 연구소로 돌아가 처음부터 다시 훈련할 수 있습니다.
